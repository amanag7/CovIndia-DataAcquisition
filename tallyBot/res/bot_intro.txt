Hi there! I'm tallyBot here to help you out with the numbers.

Below are few of the commands you can use:

`!start` : To check if I'm still up
`!help` : To get this
`!districtData` : Gives the district-wise tally
`!stateData` : Gives the state-wise tally
`!apiData` : Gives the API data of CovIndia
`!findState stateName` : Get the values of a specific state
`!findDist districtName` : Get the values of a specific district
`!stateDists stateName` : Get the values of all districts in a state
